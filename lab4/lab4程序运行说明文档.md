###编译运行方法
####2048模式
1.终端中转到lab3文件夹
2.输入

       $      make
3.输入

       $      ./game -s 2
或

       $      ./game -s 3
或

       $      ./game -s 4
或

       $      ./game -s 5
可分别设置维数为2、3、4、5的2048地图，并开始2048游戏。
       
####test（64）模式
1.终端中转到lab3文件夹
2.输入

       $     make

在路径后加入 -t 指令即可。以设置3*3的2048地图，以64为胜利标志的2048游戏为例
3.输入

       $      ./game -t -s 3
或

       $      ./game -s 3 -t
       
即可开始3*3大小、以64为胜利标志的2048游戏。
